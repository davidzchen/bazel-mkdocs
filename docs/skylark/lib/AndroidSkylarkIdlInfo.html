AndroidSkylarkIdlInfo

Provides access to information about Android rules.

Methods

  • generated_java_files
  • import_root
  • output
  • sources

generated_java_files

ImmutableCollection AndroidSkylarkIdlInfo.generated_java_files

Returns a list Java files generated from IDL sources.

import_root

string AndroidSkylarkIdlInfo.import_root

Returns the root of IDL packages if not the java root. May return None.

output

java_output AndroidSkylarkIdlInfo.output

Returns a jar file for classes generated from IDL sources. May return None.

sources

ImmutableCollection AndroidSkylarkIdlInfo.sources

Returns a list of IDL files.
