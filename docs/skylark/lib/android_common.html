android_common

Common utilities and fucntionality related to Android rules.

Methods

multi_cpu_configuration

SplitTransition android_common.multi_cpu_configuration

A configuration for rule attributes that compiles native code according to the --fat_apk_cpu and --android_crosstool_top flags.

resource_source_directory

PathFragment android_common.resource_source_directory()

Returns a source directory for Android resource file. The source directory is a prefix of resource's relative path up to a directory that designates resource kind (cf. http://developer.android.com/guide/topics/resources/providing-resources.html). May return None.
