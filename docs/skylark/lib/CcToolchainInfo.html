CcToolchainInfo

Information about the C++ compiler being used.

Methods

  • built_in_include_directories
  • link_options_do_not_use
  • sysroot
  • unfiltered_compiler_options_do_not_use

built_in_include_directories

list CcToolchainInfo.built_in_include_directories

Returns the list of built-in directories of the compiler.

link_options_do_not_use

list CcToolchainInfo.link_options_do_not_use

Returns the set of command-line linker options, including any flags inferred from the command-line options.

sysroot

PathFragment CcToolchainInfo.sysroot

Returns the sysroot to be used. If the toolchain compiler does not support different sysroots, or the sysroot is the same as the default sysroot, then this method returns None.

unfiltered_compiler_options_do_not_use

list CcToolchainInfo.unfiltered_compiler_options_do_not_use()

Returns the default list of options which cannot be filtered by BUILD rules. These should be appended to the command line after filtering.
